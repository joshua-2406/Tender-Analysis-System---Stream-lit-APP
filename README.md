# AI-Powered Tender Analysis & Location Intelligence System

Python, Streamlit, Pandas, Folium, REST APIs, NLP, OpenStreetMap (Nominatim & Overpass), PPT Automation Designed and developed an end-to-end AI-driven tender automation system that processes PDF documents, extracts structured insights, and performs location intelligence analysis. Implemented NLP-based keyword extraction for identifying project locations and integrated geocoding services to map locations dynamically across India. Developed modules for climate analysis and nearby infrastructure detection using geospatial APIs, including distance calculations via Haversine formula. Built an automated reporting system that generates professional PowerPoint presentations with summarized insights, reducing manual effort and improving operational efficiency.
